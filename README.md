# Synthetic Browsing Histories

This repository contains 500 synthetic browsing histories for 50 countries

We offer to compile more synthetic browsing histories based on our original history

More synthetic histories can be provided in a reasonable number

## Purpose

The purpose of this repository is to present extended samples of each history per country

Original published data are deposited at DOI:10.6084/m9.figshare.27979439

## Countries

Argentina, Australia, Austria, Belgium, Brazil, Bulgaria, Canada, Chile, Croatia, Czechia, Denmark, Egypt, Estonia, Finland, France, Germany, Greece, Hungary, India, Ireland, Israel, Italy, Japan, Latvia, Lithuania, Malaysia, Mexico, Morocco, Netherlands, New Zealand, Norway, Pakistan, Poland, Portugal, Romania, Saudi Arabia, Serbia, Singapore, Slovakia, Slovenia, Spain, Sri Lanka, Sweden, Switzerland, Thailand, Türkiye, Ukraine, United Kingdom, United States, Viet Nam

## Instructions

Synthetic browsing histories are in country subdirectories in [synthetic-browsing-history](synthetic-browsing-history)

Extended samples of synthetic browsing histories are shown in Jupyter notebook [history-extended-samples.ipynb](history-extended-samples.ipynb)
